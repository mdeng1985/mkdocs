    • 自然天体：小行星、行星、行星卫星、天体表面的陆标

    • AutoNav： DS1, Deep Impact (DI), Stardust

    • SMARTNav: 用于撞击小行星Didvmos卫星的导航 （see: Small-body maneuvering Autonomous Real-time Navigation (SMART NAV): Guiding a Spacecraft to Didymos for NASA’s Double Asteroid Redirection Test (DART) )

    光学导航原理和基础¶

    相机¶

    采用CCD或者CMOS探测器

    关键参数

    • instantaneous Field-of-view （IFOV). IFOV象征着角分辨率，IFOV越小，则角分辨率越高；比如：1角秒相机观测1au的陆标，1个像素的精度为750km。星敏感器之所以不适用于导航是因为它们的视场太大了。

    • 灵敏度和动态范围很重要，因为信标可能非常暗（dim)； 可检测的视觉大小可以范围从5到14，重要的是要有**相对短的曝光时间**以最大程度地减少图像的**拖尾**和**抖动**。

    以下是我的其他调研和想法：我的想法：：需要采用多个视场不同的相机（多视场多模式（多光谱）），随动机构调整指向。这样就需要实时校正指向位置。（高精度指向应该用不着） 在轨校正可以参考的文献:initial inflight calibration for hayabusa2 optical navigation camera；Updated Inflight Calibration of Hayabusa2’s Optical Navigation Camera (ONC) for Scientific Observationsduring theCruisePhase

    导航陆标选择和坐标系¶

    1. 巡航段：
      1. 选择大行星作为导航陆标的局限性：
        1. 数量少，距离远（可观性差）
        2. 亮度高，很难同时观测恒星 (同时观测恒星的作用？)
      2. 选择小行星比较合适：数量多，分布广，比如近地、Jupiter拉格朗日点附近、Centarus
    2. 环绕段
      1. 选择自然卫星合适，比如saturn的卫星。见Navigation Accuracy at Jupiter and Saturn Using Optical Observations of Planetary Satellites
      2. 选择表面的陆标也可以：
        1. 识别陆标的技术见：Characterizing and navigating small bodies with imaging data
        2. 未知陆标的技术，见Optical Navigation for Autonomous Approach of unexplored Small Bodies 小天体参数辨识 #灵感 ^f30eab
    3. 近地空间：月球表面陆标、地球同步卫星作为陆标（以在ASTERIA立方星任务中得以验证）

    巡航段的导航陆标选取算法见Kinematic Approximation of Position Accuracy Achieved Using Optical Observations of Distant Asteroids ^19271e

    相机的姿态由IMU和星敏感器得到。

    坐标系Inertial Celestial Reference Frame (ICRF) [10]：The International Celestial Reference Frame As Realized by Very Long Baseline Interferometry

    定位精度¶

    影响定位精度的因素如下： ^613344

    1. 距离陆标的距离
    2. 陆标的角距离
    3. 相机分辨率
    4. 陆标位置精度
    5. 相机姿态定位精度
    6. 陆标的中心提取精度（centerfinding)

    图像处理¶

    特征提取¶

    ^0dd3f6

    目标分为三种类型： - unresolved objects： brightness moment algorithm达到高精度的centerfinding （stars和beacon) 对应点目标？ ^085de7 - 对于深空任务而言，很容易实现。因为视场窄、相机姿态已知、陆标分布广。 - resolved objects - surface features on a body

    对unresolved objects观测识别过程如下： ^91e82f 1. 根据**航天器先验位置信息**, 计算陆标的惯性指向，然后相机指向该指向。由于航天器姿态估计误差，则该陆标在十几像素FOV范围内. \(指向=R^i-C_i^br^i\) 2. 识别Star pattern，重新计算指向 3. 调整相机姿态指向，提供更精确的指向。 4. 利用brightness moment算法找到陆标在图像中的位置。

    因此需要航天器先验信息，不适用于lost in space 问题。

    • 对于目标大于1个像素的识别，centerfinding 参考：Accurate Planetary Limb Localization for Image-Based Spacecraft Navigation
    • 对于表面陆标，则采用使用立体光度法对特征进行互相关cross-correlating the features using the stereo-photoclinometric techniques 能达到10个像素左右的精度，see:Characterizing and navigating small bodies with imaging data

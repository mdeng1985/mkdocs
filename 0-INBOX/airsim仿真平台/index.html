    airsim仿真平台¶

    airsim手把手详细教程(一)：win10配置airsim仿真环境 https://microsoft.github.io/AirSim/build_windows/

    安装¶

    主要步骤¶

    1. 参考airsim手把手详细教程(一)：win10配置airsim仿真环境下载并安装如下3个软件：

    2. 安装vs2019社区版

    3. 安装Unreal引擎安装 UE >= 4.25, however, we recommend 4.25 我下载和安装的是4.25.4 D:\Program Files\EpicGames 106g->77g

    4. 用sourcetree将airsim克隆到D:\dropbox\mdeng\WORK\MyCode\uav-sim\sourceTree-AS目录下，下载完500多M

    5. 编译airsim, 在Developer Command Prompt for VS 2019下

    cd AirSim
build cmd
    1. 测试airsim自带的block环境

    打开Developer Command Prompt for VS 2019, 然后CD 到Airsim\Unreal\Environments\Blocks文件夹下，运行update_from_git.bat。

    运行完之后，会在Airsim\Unreal\Environments\Blocks文件夹下生成Blocks.sln工程文件，双击打开，这时会打开Visual Studio 2019.

    在Visual Studio 2019中，将Block工程设为启动项目，编译选项设为DebugGame_Editor和Win64。然后点击调试->开始调试，这时就会打开Unreal Engine。

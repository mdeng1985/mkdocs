    Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors¶

    作者：Håvard Fjær Grip, Thor I. Fossen, Tor A. Johansen, and Ali Saberi 年代：2011

    本文考虑了基于多个时变向量的姿态估计，设计了观测器，估计了零偏

    基于向量的姿态估计： - [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^be089d|Mahony, Hamel, and Pflimlin]]和[[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^8fb2bc|Hamel and Mahony]]提出了基于静态参考向量的观测器，不依赖于separate attitude resolution, but instead makes use of vector measurements directly. - [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^c2def6|Hamel, Trumpf, and Lageman]] 和[[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^5c5302|Hua]]考虑了基于时变参考向量的姿态估计， 后者利用速度进行姿态估计（和我的思路有点像）

    基于单个向量(a single vector measurement)的姿态估计: - [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^c2def6|Mahony et al]]: persistency-of excitation (PE) condition满足? - [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^b51d5e|Kinsey and Whitcomb]] and [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^0e5aa8|Lee, Leok, McClamroch, and Sanyal]] [15] 没有考虑陀螺零偏 - [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^828d9a| Vasconcelos, Silvestre, and Oliveira]]同时考虑了姿态估计和陀螺零偏问题，利用参考向量的奇异值分解对测量数据进行重新对齐。 - Batista, Silvestre, and Oliveira[[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^aca3a8]] [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^041be5]] 和Grip, Saberi, and Johansen[[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^e4409e]] [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^9bc2d2]]考虑了基于时变参考向量( time-varying reference vectors)的姿态估计，但是没有考虑零偏估计问题。

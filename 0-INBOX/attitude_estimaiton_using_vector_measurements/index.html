    Typically, there are two approaches to estimate attitude using vector and gyroscope measurements: Kalman filter based approach[[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^404f67|[1]]] [[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors#^aa91fe|[2]]]and nonlinear observers[[RefNotes-Attitude Estimation Using Biased Gyro and Vector Measurements With Time-Varying Reference Vectors|[3]]]

    When gyroscope measurements are not considered, the attitude can be determined algebraiclly using multiple vector measurements [1]. To fuse vector measurements with gyroscope measurements [2][3][4], the well known Kalman filters [2][3][4] or deterministic nonlinear observers[6][7][8][9] can be applied to estimate attitude and perhaps gyro bias.

    Compared with Kalman filters, convergence properties for nonlinear observers have been proved[5]. For the case when multiple unparallled vectors are avaiable, convergence/stability properties for certain nonlinear observers have been proved to estimate attitude as well as gyro bias, no matter whether the references vectors are stationary[6][7] or not [8][9].

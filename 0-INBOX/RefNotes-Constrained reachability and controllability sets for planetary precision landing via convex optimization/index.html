    RefNotes-Constrained reachability and controllability sets for planetary precision landing via convex optimization¶

    标签¶

    阅读笔记¶

    关于任务的着陆精度¶

    ^0659dc

    火星具有稀薄的大气，典型的火星软着陆进入、下降和着陆(EDL)过程分为大气减速、伞降减速和动力减速（采用主发动机进行减速）三个过程。

    传统的火星着陆任务在大气进入段为弹道式飞行，没有轨迹控制能力，环境、气动不确定性、导航误差等不确定性带来的着陆精度为几十公里。借鉴于阿波罗的大气进入制导，2012年的好奇号任务则在火星大气进入过程采用了升力控制的制导方法，提高了大气进入段的轨迹散布，最终的着陆精度在10km左右[9]。对于基于常规着陆器的定点软着陆任务，首先需要提高大气进入段的轨迹控制精度，使得在开伞时刻的状态偏差（相对于着陆点）尽可能的小。但是由于伞降段伞本身没有轨迹控制能力，因此大气进入段的轨迹散布在伞降飞行过程中变大，这就需要在动力减速过程进行横向机动减少状态偏差，以实现定点着陆。

    有学者研究了动力下降段的解析式最优轨迹[[RefNotes-Constrained reachability and controllability sets for planetary precision landing via convex optimization#^2f9651]] [16-18].

    本文贡献¶

    求解动力下降段约束制导问题，能够保证全局最优和收敛性。采用的方法是凸优化中的内点法（保证燃料最优和着陆误差最小）[19,21]。 计算时间为10ms

    constrained reachability/controllability (CRC) sets。关键参数：推重比、燃料比例PMF， 导航不确定性以及推力器比冲。

    以往确定CRC集的方法都是采用蒙特卡洛仿真方法采样得到，这样耗时计算量大，不易于星上计算[14,15].

    本文采取的思路是将CRC集确定转化为确定其子集边界的方法，然后尽可能的增大其边界[35]。

    利用lossless convexifiction [1,19-21,37]将非凸问题转换成凸优化问题。

    将非凸问题relax成凸约束

    采用了如下的方式：

    1. 直接变换
    2. 利用变量变换将非线性微分方程转成线性微分方程
    3. 利用泰勒近似对非凸约束进行近似

    [37.]

    可达性：起始状态不固定的优化问题

    可控性：终端状态不固定的优化问题

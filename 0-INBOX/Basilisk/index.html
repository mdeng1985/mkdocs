    升级cmake¶

    需要先安装openssl ： 

    sudo apt-get install libssl-dev

    然后 

    cmake --version 
sudo apt-get remove cmake
cd ~ wget https://cmake.org/files/v3.13/cmake-3.16.0.tar.gz 
tar xvf cmake-3.13.2.tar.gz 
cd cmake-3.13.2
./bootstrap --prefix=/usr 
make 
sudo make install

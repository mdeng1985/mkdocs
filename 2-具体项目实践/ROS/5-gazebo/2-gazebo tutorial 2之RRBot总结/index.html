    2-gazebo tutorial 2之RRBot总结¶

    在Rviz中显示¶

    运行如下命令: roslaunch rrbot_description rrbot_rviz.launch 该launch文件中的主要原理是运行rviz包的rviz节点,调用的是rrbot.rviz文件, 其中该文件中的参数robot_description里给出了机器人的模型rrbot.xacro,该参数是传递给rrbot.rviz

    在Gazebo中显示¶

    运行如下的命令 

    roslaunch rrbot_gazebo rrbot_world.launch

    rrbot_world.launch中的主要流程 - 将rrbot.world世界文件传递到$(find gazebo_ros)/launch/empty_world.launch中显示world - rrbot.xacro赋值给robot_description参数 - 运行spwan_model节点生成机器人, 该机器人模型由robot_description给出.

    运行完以后会在Gazebo显示该模型, 并会随着重力的作用运动.

    在Gazebo里控制机器人运动¶

    • 首先启动仿真 
      roslaunch rrbot_gazebo rrbot_world.launch
    • 然后启动控制器 
      roslaunch rrbot_control rrbot_control.launch
    • 给控制器发送命令就可以控制机械臂运动了 
      rostopic pub -1 /rrbot/joint1_position_controller/command std_msgs/Float64 "data: 1.5"
rostopic pub -1 /rrbot/joint2_position_controller/command std_msgs/Float64 "data: 1.0"

    更详细的解读见[[3-Tutorial ROS Control#^ae67a5|RRBot 实践]]

    添加可乐瓶并控制运动¶

    • 首先启动仿真 
      roslaunch rrbot_gazebo rrbot_world.launch
    • 添加一个可乐瓶模型到场景中 
      rosrun gazebo_ros spawn_model -sdf -database coke_can  -model coke_can -y 1
    • 利用rostopic发布器,设置可乐瓶的初始状态 
      rosservice call /gazebo/set_model_state '{model_state: { model_name: coke_can, pose: { position: { x: 0.3, y: 0.2 ,z: 0 }, orientation: {x: 0, y: 0.491983115673, z: 0, w: 0.870604813099 } }, twist: { linear: {x: 0.0 , y: 0 ,z: 0 } , angular: { x: 0.0 , y: 0 , z: 0.0 } } , reference_frame: world } }'

    • Gazebo会发布model_states和link_states输出模型或者模型里的link状态,因此可以查看这些状态: 

      rostopic echo -n 1 /gazebo/model_states
rostopic echo -n 1 /gazebo/link_states

    • 利用rosservice call对rrbot模型进行设置,rrbot在初始状态和重力的作用下会运动, 踢飞可乐瓶 

      5. 

```bash
rosservice call /gazebo/set_model_state '{model_state: { model_name: rrbot, pose: { position: { x: 1, y: 1 ,z: 10 }, orientation: {x: 0, y: 0.491983115673, z: 0, w: 0.870604813099 } }, twist: { linear: {x: 0.0 , y: 0 ,z: 0 } , angular: { x: 0.0 , y: 0 , z: 0.0 } } , reference_frame: world } }'

    • 可以查看可乐瓶的状态 

      rosservice call gazebo/get_model_state '{model_name: coke_can}'

    • 对可乐瓶施加力矩 

      rosservice call /gazebo/apply_body_wrench '{body_name: "coke_can::link" , wrench: { torque: { x: 0.01, y: 0 , z: 0 } }, start_time: 10000000000, duration: 1000000000 }'

    • 得到仿真世界和目标的属性rosservice call gazebo/get_world_properties

    • 删除模型rosservice call gazebo/delete_model '{model_name: coke_can}'

    Linux常用命令¶

    标签¶

    linux¶

    用户切换¶

    • 普通用户切换到root | sudo su. 或者sudo -i
    • root切换到用户ubuntu: su ubuntu

    软件安装卸载、系统清理等¶

    更新软件源¶

    软件源的地址位于/etc/apt/sources.list，在修改地址以后可以采用 apt-get update对软件包列表进行

    更新已安装软件apt-get upgrade¶

    ### 安装一个新软件包apt-get install packagename

    ### 卸载软件包 - apt-get remove packagename （保留配置文件）
    - apt-get --purge remove packagename（删除配置文件）
    - sudo apt-get autoremove

    sudo apt-get remove --auto-remove python # 卸载python3及其依赖,慎重使用，可能把其他的包(比如ros)也给卸载了

    清理磁盘空间¶

    定期运行这个命令：apt-get autoclean来清除那些已经卸载的软件包的.deb文件
    参考资料 http://man.linuxde.net/apt-get
    从 0 开始学习 Linux 系列之「01.安装 Linux 软件的 3 种方法」

    文件操作¶

    调出终端快捷键:ctrl+alt+t¶

    寻找某个文件¶

    find / -name "xorg.conf.d"

    查看文件相关¶

    • 查看当前目录pwd
    • 查看当前目录下的文件文件夹ls -lt
    • 查看文件大小ls -hl
    • 查看所有文件ls -a（可以显示隐藏文件，比如.bashrc)
    • 查看帮助： man mkdir 查看mkdir命令的帮助
    • 显示隐藏文件夹: ctrl+h

    复制粘贴¶

    • cp ~/test .将test拷贝到当前目录

    文件编辑¶

    nano命令
    如果你要保存所做的修改，按下Ctrl+O。想要退出，按下Ctrl+X From https://www.cnblogs.com/haichuan3000/articles/2125943.html 对文件进行编辑

    创建软链接¶

    ln -s 源地址 目的地址

    比如我把linux文件系统rootfs_dir软链接到/home/jyg/目录下

    ln -s /opt/linux/rootfs_dir /home/jyg/rootfs_dir就可以了

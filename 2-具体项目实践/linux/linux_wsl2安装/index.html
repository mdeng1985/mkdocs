    安装¶

    1. 切换到inside版本 (Windows Build version 18917 or higher)
    2. 安装wsl，参考https://docs.microsoft.com/zh-cn/windows/wsl/install-win10
      1. 管理员身份打开powershell,输入：dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart并重启
      2. 管理员身份打开powershell，输入:dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart并重启
      3. 下载wsl2更新包：适用于 x64 计算机的 WSL2 Linux 内核更新包安装
      4. 设置wsl2：wsl --set-default-version 2
    3. 进入microsoft商店下载并安装ubuntu(如果无法加载页面的，需要在ie里的internet里关闭代理）
    4. 升级成wsl2，参考https://docs.microsoft.com/en-us/windows/wsl/wsl2-install
      1. 首先在powershell里查看wsl -l的版本得到Ubuntu-18.04
      2. wsl --set-version Ubuntu-18.04 2
      3. wsl --list --verbose进行确认

    如果安装的时候提示已安装此应用，则需要参考重新初始化microsoft store

    修改软件源¶

    • 先查看ubuntu的版本sudo lsb_release -a
    • 进入cd /etc/apt
    • 打开sources.list： sudo nano sources.list
    • 将软件源替换掉,https://mirror.tuna.tsinghua.edu.cn/help/ubuntu/ 给出的软件源
    • sudo apt-get update
    • https://sspai.com/post/43813 介绍了如果使用vim修改软件源

    也可以用vim进行替换 - 先删除rm /etc/apt/sources.list - 再新建一个sudo vim /etc/apt/sources.list - 更新完以后sudo apt-get update

    vim的几个操作命令：

    • 在命令模式的时候，按i进入编辑，然后复制粘贴。
    • esc切换到命令模式，按:wq进行保存

    WSL ssh登录及远程编译设置¶

    使用ssh工具连接到ubuntu on windows（wsl）

    sudo apt-get remove openssh-server
sudo apt-get install openssh-server
sudo vi /etc/ssh/sshd_config
    然后 
    Port 22 #默认即可，如果有端口占用可以自己修改
PasswordAuthentication yes # 允许用户名密码方式登录


    sudo service ssh restart

    然后xshell登录看看，主机名为localhost

    安装windows terminal¶

    首先下载 windows terminal ：
    https://github.com/microsoft/terminal/releases

    下载的文件是 msixbundle 类型的文件，所以需要 powershell 中输入以下命令进行安装：

    Add-AppxPackage ./[文件名xxx].msixbundle

    必须要使用 powershell 而不能使用 cmd ，因为 Add-AppxPackage 命令只在 powershell 中存在。

    安装完成后，打开 windows terminal ， 在新建窗口旁边的下拉菜单里面可以选择 ubuntu 系统了。

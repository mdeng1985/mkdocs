    Sphinx笔记¶

    注释¶

    linux下安装¶

    基本安装¶

    1. 安装
        sudo apt-get install python3-sphinx
    sudo apt-get install python3-pip

    1. 然后创建一个文件夹，用于放文档。在该目录下的bash下输入sphinx-quickstart,如果提示语言则输入 

          zh_CN
      提示后缀名的话就输入md

    2. 最后输入make html就在build目录下生成网页了。

    设置支持markdown文件¶

    一种方法是安装以下的两个工具包，支持markdown 

        pip3 install --upgrade recommonmark
    一定要有--upgrade，否则我安装完以后make html会出错。

    如果编译出错，我又卸载掉了上面的工具包和python3-sphinx，重装了python3-sphinx。

    然后修改conf.py: 

    extensions = ['sphinx.ext.autodoc',
    'sphinx.ext.doctest',
    'sphinx.ext.intersphinx',
    'sphinx.ext.todo',
    'sphinx.ext.coverage',
    'sphinx.ext.mathjax',
    'sphinx.ext.ifconfig',
    'sphinx.ext.viewcode',
    'recommonmark'
    ]
source_suffix = {
    '.rst': 'restructuredtext',
    '.txt': 'markdown',
    '.md': 'markdown',
}

    主代码index.md 

    # Welcome to TryIt's documentation

## test

[about](../about.md)


Indices and tables
==================

* :ref:`genindex`
* :ref:`modindex`
* :ref:`search`

    然后继续make html就可以了

    安装主题¶

    pip3 install --upgrade sphinx_rtd_theme
    修改conf.py 
    extensions = ['sphinx.ext.autodoc',
    'sphinx.ext.doctest',
    'sphinx.ext.intersphinx',
    'sphinx.ext.todo',
    'sphinx.ext.coverage',
    'sphinx.ext.mathjax',
    'sphinx.ext.ifconfig',
    'sphinx.ext.viewcode',
    "sphinx_rtd_theme",
    'recommonmark'
    ]
html_theme = "sphinx_rtd_theme"

    其他还不错的主题:

    • asteroid-sphinx-theme
    • ydoc-theme
    • sphinx-zon-theme
    • Caktus

    在index.rst里添加 

    .. toctree::
             :numbered:
           :maxdepth: 1
           :caption: Contents:

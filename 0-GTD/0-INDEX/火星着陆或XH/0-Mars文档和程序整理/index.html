    1. 首先运行仿真XH主程序.txt （Mars-EDLGNC并行计算目录），在上级目录下的result生成如下轨迹：
      • 单次仿真轨迹：lmdFDIRMC\single_1.dat（由dataLogMe读取）
      • 单次仿真轨迹：lmdFDIRmc\singleDyn_1.dat （由dataLogMe1ms读取，产生4ms的动力学轨迹）
      • 打靶轨迹：data_*.dat （由matlabMonteCarlo进行读取）
      • 打靶轨迹：datalmd_*.dat （由matlabMonteCarloLMD进行读取）
    2. MATLAB数据复现和分析，将数据拷贝到MatlabAnalysis文件夹下进行数据读取和分析，具体包括：
    3. 轨迹复现文件夹：
      • 读取c生成的动力学数据，模拟陀螺数据和测距测速敏感器，进行后台滤波

    4. 引力修正程序读取lmdFDIRMC\single_1.dat（相当于遥测数据），并在matlab进行后台滤波，和星上的比对。

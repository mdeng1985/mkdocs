    硬件配置和重量预算¶

    ​

    每个小的着陆器的硬件组成如下： - GNC敏感器：参考火星无人机的配置：IMU、nadir pointed相机、time-of-flight optical rangefinder [[Titan Aerial Daughtercraft]] 总共不到500g - 热防护罩 REBR 3kg 见[[RefNotes-MarsDrop#重量]] - 伞 （滑翔伞）预算为3kg,对应\(6000cm^3\)内部体积，见[[RefNotes-MarsDrop#伞技术对比]]， Marsdrop选择的伞的重量为1.5kg（对应的着陆器的重量才3kg，其中1kg是载荷） - 终端减速机构（airbag或者推力器) - 太阳帆板 - UHF天线 - 载荷 (参考[[RefNotes-The Small Mars system]]) - 无人机 （7kg） - 尘埃分析仪 (600g) - 天气传感器、麦克风、地震仪 这些科学仪器不超过1kg - 其他：比如：PUFFER

    https://wenku.baidu.com/view/d3a3426ca45177232f60a28b.html 微型推力器

    初步计划所有重量不超过20kg，着陆器10kg，伞4.5kg, REBR热防护罩3kg, 实际上可以参考[[RefNotes-MarsDrop#^260ed5]]（包括伞和防护罩3.5kg）, 再此基础上加上PUFFER或者无人机。然后适当对伞和防护罩放大点。

    Marsdrop着陆时垂向速度7m/s，水平速度18.7m/s， 再加上airbag刚好[[ESA – Airbags for Small Landers]]， 也即Design of an Airbag Landing System for Small Mars Landers 16kg

    发展微型着陆器的必要性和优势¶

    现有大型探测器只是部件级冗余，对于高风险和高回报的探测，不能容许失败的风险。[[RefNotes-The Shapeshifter a Morphing,Multi-Agent, Multi-Modal RoboticPlatform for the Exploration of Titan#现有探测方式的不足]]

    其余 [[RefNotes-MarsDrop#必要性]]

    优势： 1. 成本低 （现有方式的5%，见[[RefNotes-MarsDrop]]） 2. 风险小 3. 可扩展性好，能够用于其他有大气天体 比如venus， titan等。（由于成本低，可以放心在其他环境更为恶劣的天体上试验） 4. 同时多目标探测，降低探索周期。不用先开展前哨任务。同时获取多个目标的科学数据，可以用于周期性事件的监测。 5. 能够实现机遇探测。 6. 能够实现定点着陆。 7. 探索高风险区域（添加一个图，见[[RefNotes-MarsDrop#必要性]]）, [[RefNotes-MarsDrop#^972d63]]一文中进行了详细阐述。 8. 能够和高校和私营机构合作 9. 可以作为前哨任务，见[[RefNotes-MarsDrop#^e25f78]]

    搭载方式方便： 1. 可以同时搭载多个 2. 也可以作为伴随飞行器搭载在主飞行任务上。

    删除的部分¶

    ​

    火星大气进入、下降和最终着陆(Entry, Descent, and Landing, EDL)过程是软着陆任务最为重要和关键的部分之一。这一过程导航系统的设计是组网式微型软着陆任务的关键技术之一，为了设计可靠性高、性能好的自主导航系统必须考虑如下几个重要问题：

    相比传统中大型软着陆探测器，微型软着陆探测器机动能力有限，需要进行可达性分析，考虑环境气动等不确定性，设计合适的标称轨道，以满足有限区域定点着陆能力。

    传统软着陆探测方式在抛大底以前主要依赖于惯性导航进行抛伞、抛大底等关键事件触发，由于微型敏感器系统误差特性复杂、稳定性差等特性，这些事件触发方法难以直接移植到微型探测器上，因此需要研究基于微型敏感器的EDL关键事件触发新方法。

    微型敏感器精度低，如何针对复杂的火星EDL飞行环境，对多个低精度信息源进行有效融合，并实现每个探测器相对于各自着陆点的精确自主导航

    微型敏感器可靠性差，需要研究高可靠鲁棒融合自主导航方法，实现高容错的着陆自主导航。

    ​

    针对微型着陆器

    组网式火星微型探测器软着陆任务能够实现对常规软着陆探测任务缺点的优势互补，有着很大的应用前景。导航系统的设计是组网式微型软着陆任务的关键技术之一，相比于常规的软着陆探测任务，组网式微型软着陆任务的自主导航有着自身的难点和亟需解决的问题：

    (1) 对于组网式探测器，探测器之间可以实现相互星间测量，因此某个探测器本身还充当着其他探测器的导航源的角色，由多探测器构成的导航信息源空间运动几何如何最优布局，用以提高导航可观测性是一个新问题。

    (2) 受微型探测器自身体积、重量和功耗的约束，携带的敏感器只能为微型敏感器。微型敏感器的误差特性复杂、稳定性差，因此需要在整个进入、下降和着陆过程进行实时校正是组网式微型探测器面临的另外一个问题。

    (3) 在算法层面，针对火星进入、下降和着陆复杂的飞行环境，并考虑微型探测器自身计算和存储能力的限制，如何基于低成本的多类型微型敏感器组合构建高性能实时的自主导航算法是亟需解决的问题。

    项目基本信息¶

    项目名称：基于微型探测器的火星定点着陆自主导航与制导方法研究

    英文：Research on autonomous navigation and guidance methods of Mars miniature soft landing missions

    摘要： 基于火星微型探测器的软着陆任务是对现有软着陆任务的有效补充，可以实现多探测器多区域同时探测，对实施低成本获取多信息有潜在的价值。自主导航与制导是实现火星微型探测器软着陆的关键技术之一。本项目针对以此为背景，研究火星微型探测器软着陆自主导航与制导方法，重点研究基于性能分析的制导及导航信息源配置方法、微惯性着陆自主导航方法以及基于微导航敏感器的多源信息融合自主导航算法等问题，从理论方法层面突破火星微型探测器软着陆自主导航与制导的核心技术，项目研究成果直接应用于火星低成本着陆任务，并可推广到其他有大气天体的低成本软着陆任务。

    英文摘要： Mars soft landing mission using micro-probes is an effective complement to current soft landing missions. A network of micro-probes can be deployed across a wide area to realized multi-region exploration, which has potential value for implementing low-cost access to multiple information. Autonomous navigation and guidance methods are one of the key technologies for Mars soft landing missions using micro-probes. This project studies the soft landing autonomous navigation and guidance of the Mars micro-landing, focusing on the guidance and navigation beacon source configuration based on performance analysis, MEMS-based inertial navigation for mars landing, and MEMS sensors based multi-source information fusion autonomous navigation algorithm. This project tries to break through the core technology of the autonomous navigation system for Mars micro-landing missions from the theoretical view. The results can be directly applied to the Mars low-cost landing missions, and can further be applied to other low-cost soft landing missions for atmospheric celestial bodies.

    关键词：火星探测，深空探测，自主导航，微型着陆器， 定点着陆 mars exploration, deep space exploration, autonomous navigation, micro lander, pinpoint landing

    科学问题¶

    ​

    在自主控制技术的大方向下，提炼亟需解决的科学问题，从“自主导航、自主制导与控制、自主任务规划、自主故障诊断与重构”四个方面深入开展基础性、创新性的科学技术研究和核心技术攻关，力争突破航天器长期自主安全运行面临的技术瓶颈问题和基础难题。

    探索浩瀚宇宙、和平利用太空，是全人类的共同梦想。以嫦娥四号任务圆满成功为标志，我国探月工程四期和深空探测工程将全面拉开序幕，后续重点开展月球永久阴影区探测、小行星采样返回探测、火星采样返回探测、木星系及行星际穿越探测、太阳系边际探测等一系列深空探测活动，研究太阳系起源与演化、地外生命信息探寻等重大科学问题。

    与地球轨道航天器相比，深空探测器飞行距离更远、运行时间更长、任务环境更复杂，传统的“地面测控站—航天器”大回路操作控制模式已难以满足高精度、强自主的实际需求，如何保证航天器长期自主安全运行是后续深空探测任务实施亟需解决的技术瓶颈问题。自主控制技术是解决这些问题的有效途径，一直是国内外深空探测领域研究的热点和难点。

    深空探测自主控制技术是通过在航天器上构建一套精度高、自主性强、安全可靠的自主运行管理系统，实现长时间无人参与情况下的自主安全运行，具体任务是根据飞行阶段和周围环境，自主开展工程任务与科学任务的调度规划、感知导航、命令执行、器上状态监测与故障时的系统重构，确保实现深空探测器的安全可靠自主运行。

    本创新群体将紧密围绕深空探测国家重大需求，以后续规划的火星采样返回、木星系及太阳系边际探测等一系列任务为牵引，在自主控制技术的大方向下，提炼亟需解决的科学问题，从“自主导航、自主制导与控制、自主任务规划、自主故障诊断与重构”四个方面深入开展基础性、创新性的科学技术研究和核心技术攻关，力争突破航天器长期自主安全运行面临的技术瓶颈问题和基础难题。

    探索浩瀚宇宙、和平利用太空，是全人类的共同梦想。"天问一号"火星探测器的成功发射拉开了我国行星际探测的序幕。火星探测任务的大气进入、下降和着陆(EDL)过程（俗称恐怖七分钟）是现有深空探测任务难度最大的任务阶段，也是影响着陆任务成败最为关键的阶段。现有火星着陆探测任务的成功率不到50%，其中EDL过程中的技术故障是引起着陆任务失败的主要原因。现有基于大型着陆器的火星软着陆任务探测存在着范围有限、受载荷能力限制着陆器机构复杂，成本高、着陆难度大、风险大等问题，因此迫切寻求新的任务方式。

    本项目将紧密围绕深空探测国家重大需求，以火星探测为牵引，寻求以小成本任务为背景的火星着陆任务，并重点解决基于多微型着陆器的自主导航问题，开展创新性科学研究，为未来火星低成本着陆任务的顺利实施奠定基础。

    经费预算¶

    小帖士预算N01:各项支出预算比例—1、设备费：一般购置及改造租赁0；设备试制费（可外协）根据实际需要报，全部外协一般不超50%，设备费预算不予调增；2、材料费软课题一般为0，工材学部等相关课题研究确有需要的根据实际需求填报，获资助后确保可联系物资领取入账；3、测试化验加工费体现直接研究经费的支撑性，根据实际研究需要填报，全部外协一般不超50%（测试与设备试制为可外协支出，总量一般不超50%）4、燃料动力费0；5、差旅/会议/国际交流 10-30%，此项超10%的需提供测算依据；6、出版/信息/文献 一般10-20%；7、劳务费不超10%（基金委预算评审反馈意见建议体现）；8、专家咨询费不超10%（附：以上比例为根据历年批复资助经费及验收课题的实际决算情况给出的，申请人根据实际需求可酌情增减）。

    小帖士预算N02（续）：《预算说明书》编报要点及建议格式：对各项支出的主要用途和测算理由以及合作研究外拨资金和单价大于等于10万元的设备费进行说明，鉴于基金委审核重点“目标相关性、政策相符性、经济合理性”，建议如下：首先用一小段话简单概括项目主要研究内容，体现目标相关性（让评审专家知道你干什么，不用翻本子）；接下来本项目申请经费#万元（所有单位统一为万元），各项支出的主要用途和测算依据如下：；然后分科目写说明，说明要充分，如设备试制费和测试加工费等大额支出项，可分模块进行说明，用于哪些技术名称（一定与研究内容相关）测试模块、加工等，设备分模块小于10万；其他各项说明严格依据《预算表编制说明》中的用词，如圈注，体现政策相符性；各项测算依据，主要会议费、差旅费等列等式：#万元/人次*#人次=#万元，或#万元/年*#研究年=#万元，注意测算的正确性和合理性有合作研究单位的需要提前联系合作方编报分预算，并由负责人签字后提交科技处备查，申请人汇总后统一编报，并在说明书中体现各家经费和比例。

    其他注意事项

    问题：
1、设备试制费部分，前两条列出的探针有通用设备之嫌，可不予突出，避免找问题，可与第3条整合，低于10万元的情况，可不必拆分模块，直接写明试制设备名称即可，本身经费就不多；
2、差旅会议部分算测算依据算式，左侧更新为元为单位；
3、出版/文献写法更新：论文版面费合计*万元，专利申请费合计*万元；
4、劳务费可不预算，目前所内不能支出；
5、专家咨询费写法更新，用于项目研究过程及成果的相关咨询费用，合计*万元

    综合而言，预算说明书是一个有机整体，作为申请书的一部分，评委虽然不会因为这一部分枪毙，但也请申请人重视，编制原则重点体现三性：
目标相关性
政策相符性
经济合理性
则：
1、大额支出务必写明需求，体现相关性；
2、核心关键技术不能外协；（起码不能列出来！）
3、标准尽量通用，体现政策相符，经济合理
整体应简洁明了，可读性强，能让评委觉得必须给你这些支撑，不给觉得不行是高手！

    Markmom 实践¶

    安装与配置¶

    步骤如下：

    • 安装pandoc （安装文件夹放到环境变量）
    • 安装node.js (我把安装文件夹也放到环境变量了)
    • cmd下运行命令npm install -g markmon
    • sublime 3 的package control里安装markmom
    • sublime 3 的package setting的markmom的用户配置(`Preferences > Package Settings > markmon > Settings - User｀)进行如下设置：
         {
  //On Windows try "markmon.cmd" if you get errors.
  //If markmon is not on your path you'll need to use a full path to it
  "executable": "markmon.cmd",
  "port": 3002,
  "command": "pandoc -s --mathjax --toc --standalone --template=d:/dropbox/mdeng/someBackup/pandocTemplate/theorem-number-html/templates/pandoc-amsthm.html --number-sections --filter=pandoc-crossref --filter=pandoc-citeproc --filter=d:/dropbox/mdeng/someBackup/pandocTemplate/theorem-number-html/py/pandoc-amsthm.py --bibliography=d:/Dropbox/mdeng/WORK/knowldegBase/refAll.bib -t HTML5",
  "stylesheet": "d:/dropbox/mdeng/someBackup/pandocTemplate/template/css/github-pandoc.css",
  "projectdir": null
  }
    • tools-markmom-launch
    • 或者设置预览快捷键Preferences → Package Settings → Markmon → Key Bindings - User
    [
    { "keys": ["ctrl+shift+m"], "command": "markmon_toggle", "args":
        {
            "enable": true
        }
    }
]

    markmon的设置说明:

        Usage: node ./bin/markmon [filename] --port [num] --command [string] --view [string] --projectdir [path] --stylesheet [path] --help

    Options:
      --port        Port to listen to                                                 [default: 3000]
      --command     Command to parse markdown to html                                 [default: "pandoc --mathjax -N -t HTML5"]
      --view        Command to execute after the server is setup
      --projectdir  Root directory of your project, useful for local image resources
      --stylesheet  Path to your custom stylesheet
      --help        Get this help message

    最终的设置如下：

         {
  //On Windows try "markmon.cmd" if you get errors.
  //If markmon is not on your path you'll need to use a full path to it
  "executable": "markmon.cmd",
  "port": 3002,
  "command": "pandoc -s --mathjax --toc --standalone --template=d:/dropbox/mdeng/someBackup/pandocTemplate/theorem-number-html/templates/pandoc-amsthm.html --number-sections --filter=pandoc-crossref --filter=pandoc-citeproc --filter=d:/dropbox/mdeng/someBackup/pandocTemplate/theorem-number-html/py/pandoc-amsthm.py --bibliography=d:/Dropbox/mdeng/WORK/knowldegBase/refAll.bib -t HTML5",
  "stylesheet": "d:/dropbox/mdeng/someBackup/pandocTemplate/template/css/github-pandoc.css",
  "projectdir": null
  }

    几点说明：

    1. 最好是确保pandoc-crossref和pandoc -citeproc的exe文件和pandoc.exe在同一个文件夹内

    2. --bibliography不能是缩写--biblio，路径分隔符为/不是\

    3. 有时候会提示pandoc打开文件不存在，这时候需要将命令的空格符删掉，再来一次。

    4. 要生成toc目录，则需要--toc --standalone。

    6. -s --mathjax 里的-s 不能少，否则每次刷新公式会错位。

    vscode的pandoc插件¶

    安装完以后，在preferences-settings里搜pandoc，进行如下的设定

    {
    "workbench.colorTheme": "Default Light+",
    "pandoc.htmlOptString": "--toc --standalone --number-sections --filter pandoc-crossref  --filter pandoc-citeproc  --bibliography=d:/Dropbox/mdeng/WORK/knowldegBase/refAll.bib -s --mathjax --css=d:/dropbox/mdeng/someBackup/pandocTemplate/template/css/github-pandoc.css 
    --highlight-style=kate --template=D:/Dropbox/mdeng/someBackup/pandocTemplate/theorem-number-html/templates/pandoc-amsthm.html",
}

    然后按ctrl+k,再按P键就可以了。

    公式相关¶

    • 编号和引用

    所需要的插件：markdown-preview-enhanced和pandoc(电脑直接安装)。 还是借助于pandoc的--filter=pandoc-crossref(用法参考pandoc.md)。

    markdown-preview-enhanced的设置如下：

    • file-setting-package在这个package下设置
    • 选中using pandoc parse
    • options里写入：--filter=pandoc-crossref

    demo如下

    $$ \begin{matrix} a=3 \end{matrix} $$ {#eq:a3} [@eq:a3]

    • 黑体

    \newcommand{\bm}[1]{\boldsymbol #1}然后\bm，比如： \(\bm{a}\)

    引用文献¶

    主要参考pandoc.md - 文献数据库为ref.bib - md文件中引用方法为[@hinton2006fast],其中@后为bibtex的key - pandoc命令：--filter pandoc-citeproc --biblio ref.bib但是markdown-preview-enhanced里的设置为--filter=pandoc-citeproc,--biblio=ref.bib，要用等号和逗号进行隔开。

    快捷键¶

    • 搜索命令的快捷键cmd-shift-P
    • 预览cmd-shift-M
    • cmd-\ 显示或隐藏目录树
    • ctrl-0 焦点移到目录树
    • fn-F2(选择tree后) 修改文件/文件夹名称
    • ctrl-shift-M Markdown预览，这非常方便俺这种写手
    • ctrl-alt-b 安装atom-beautify后可使用，格式化代码
    • ctrl-` 安装terminal-panel后可使用，调起CLI
    • ctrl-shift-U 调出切换编码选项
    • cmd-t或cmd-p 查找文件
    • alt-cmd-[ 折叠
    • alt-cmd-] 展开
    • alt-cmd-shift-{ 折叠全部
    • alt-cmd-shift-} 展开全部
    • ctrl-m 相应括号之间，html tag之间等跳转
    • alt-shift-S 查看当前可用代码片段
    • alt+shif-e 多标签查看(需要安装expose插件)

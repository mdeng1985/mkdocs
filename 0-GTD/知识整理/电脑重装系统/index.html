    安装dropbox (目前用不着)¶

    • 代理选择：socks5 127.0.0.1 1080

    由于我的d盘里已经有了dropbox文件夹，因此需要如下设置：

    • 将d盘里的dropbox文件夹重命名为dropbox_
    • dropbox首选项----同步----dropbox文件夹--点移动-然后选则D盘(dropbox的上层目录)
    • 将生成的dropbox文件夹删除，然后将原来那个dropbox_又重命名为dropbox

    另外一种解决方案： 

        1. Select the Dropbox folder location, if not in the default location.
        ○ Important: Because this process removes the previous settings and configuration, if you wish to avoid re-downloading your Dropbox content you will need to manually choose the location for Dropbox to sync to. If you don't manually choose a location, you'll create a new Dropbox folder and start to download your content again.
        1. To select your established Dropbox folder on your drive, click Advanced Settings after choosing which folders to sync.  （主要是这一项）
    Then choose the parent folder of your already established Dropbox folder. For example, if you have your Dropbox folder in the path D:\OtherDrive\Folder\Dropbox, select the folder D:\OtherDrive\Folder.

    最新更新，本地有dropbox文件夹，同步可能会冲突，解决方案如下： 1. 将本地的dropbox文件夹重命名为dropbox_ 2. 安装dropbox,然后选择性同步-要选择本地dropbox_包含的文件夹（比如mdeng），然后设置为在线访问 3. 然后进行同步（同步完成以后） 4. 利用freefilesync对dropbox_和dropbox进行比对更新， 将mdeng再设置为本地访问

    Base2Tone-me.css后面lmd注释后面的。 包括 - 对页边距调整 - list对齐问题

    /* remove the yaml metadata from embed */

/* Remove embed yaml first separator */
.markdown-embed-content > hr:first-child {
  display: none;
}

.markdown-embed-content > hr:first-child + p {
  display: none;
}

/* Remove embed yaml second separator (if empty) */

  display: none;

/* Remove embed yaml second separator */
.markdown-embed-content > hr:first-child + p + hr {
  display: none;
}

/*降低页边距*/

  padding-left: 0;

.markdown-source-view {
  padding-right: 0px;
}

.markdown-preview-view {
  max-width: 750px;
  text-align: left;

  padding-inline-start: 18px;

    五线谱渲染music-notion¶

    X:1
T:The Legacy Jig
M:6/8
L:1/8
R:jig
K:G
GFG BAB | gfg gab | GFG BAB | d2A AFD |
GFG BAB | gfg gab | age edB |1 dBA AFD :|2 dBA ABd |:
efe edB | dBA ABd | efe edB | gdB ABd |
efe edB | d2d def | gfe edB |1 dBA ABd :|2 dBA AFD |]
    Chorus music 和弦 
    X: 1
T: Chorus
V: T1 clef=treble name="Soprano"
V: T2 clef=treble name="Alto"
V: B1 clef=bass name="Tenor"
V: B2 clef=bass name="Bass"
L:1/8
K:G
P:First Part
[V: T1]"C"ed"Am"ed "F"cd"G7"gf |
[V: T2]GGAA- A2BB |
[V: B1]C3D- DF,3 |
[V: B2]C,2A,,2 F,,2G,,2|

    注意语法： 1. L ⅛表示最小单位为⅛音符，这时候C1渲染成⅛音符; L ¼表示最小单位为¼音符，这时候C1渲染成¼音符 2. 对于高音谱号，C为中央C，小写的c，则高8度 4. "1"可以显示指法 5. ^F表示升F, 6. 降调To flatten a note precede it with an underscore _B表示降B

    calendar插件+day planner实现日程管理¶

    1. 安装day planner和calendar插件
    2. day planner设置里day planner mode里设置为command mode
    3. calendar面板内打开创建今天的日记，然后在控制面板里选择day planner: link today's day planner to the current note

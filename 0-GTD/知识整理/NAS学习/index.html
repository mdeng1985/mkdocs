    NAS学习¶

    需要下载的软件¶

    1. Qfinder Pro 让 Windows、Mac 及 Ubuntu 使用者快速搜寻、安装及存取局域网内的 QNAP NAS。使用Windows 版本的「Storage Plug & Connect」功能，更可将 NAS 当作联机的网络驱动器或是虚拟磁盘，方便访问档案。 ^415b2c
    2. Qsync 可自动将档案同步至与 QNAP NAS相连的装置上。当您将档案拖拉至 Qsync 文件夹后，只要任何信息有异动，所有与NAS 相连的装置皆会同步更新档案。智能同步及进阶的选择性同步可让你选择只同步 Qsync 文件夹中特定的子文件夹。除此之外，您可以建立下载链接与他人分享档案，大幅省去建立账号密码的时间，让档案分享更便利。
    3. Hybrid Backup sync 3
      1. HBS 3 帮助您有效率地备份文件至本地端 NAS、外接装置 (USB 一键启动备份功能)、远程服务器或云端存储空间中；多版本备份机制，确保重要数据妥善保存，更可让您便利追踪修改纪录。

    手机照片、视频音频备份到NAS上¶

    分为如下四步

    • 手机上Qfile自动上传到NAS上的某个文件夹A
    • NAS利用Qfiling进行整理，整理到另外一个文件夹B
    • NAS上利用某个软件或者手动再整理到C文件夹（公用的，目前这一步还未开展）
    • NAS的photo station/video station添加文件夹B、C
    • 手机端Qphoto/Qmusic等可以进行照片的查看（查看的是B文件夹和C文件夹）

    具体操作如下：

    手机 手机端目录 NAS文件夹A NAS文件夹B NAS文件夹C
    三星NOTE8 自动设置+音频目录 NAS/photos/mdPhone1 NAS/photos/sort NAS文件夹C
    岳欣IPHONE NAS/photos/mdPhone1 yuexin/photos/iphoneUpload NAS文件夹C

    TIPS

    • 三星的音频目录为/storage/emulated/0/voice Recorder
    • C文件夹暂定multimedia/photos
    • Qfile上传完文件会显示上传了多少文件，然后进入到手机自己本身的相册上，看看是有多少文件，确认无误以后，删除手机中的文件
    • Qfiling整理的几点注意事项：
      • 不要重命名，这样后面不好搜索重复文件
      • 规则如下：
      • ![[qfiling_rule.png]]
    • photo station/video station添加文件夹的时候，该文件夹必须启用了共享文件可同步化功能，否则是看不到的。FileStation里可以进行查看

    webdav共享zotero的资料库¶

    1. file station里设置显示隐藏文件夹 （右上角-三个点里设置）
    2. 下载HSB3, 选同步---单项同步作业---本地NAS
      1. 源文件夹为：homes/mdeng1985/.Qsync/WORK/zotero-资料库
      2. 目标文件夹为:NAS/webdav/zoteroLib ![[hbs3_zotero.png]]
    3. qsync central -共享文件夹-启用NAS文件夹
    4. 接下来开启webdav服务
      1. 控制台-应用服务-web服务器-启用webdav（选中Webdav权限），设置http端口号为5007
      2. 控制台-权限-共享文件夹-修改NAS的共享文件夹权限，设置成mdeng1985账户是无限制的访问webdav
    5. IPAD PAF EXPORTER
      1. http://192.168.1.110:5007/NAS 设置mdeng1985及其密码即可。(NAS区分大小写) (也可以外网访问：http://221.219.69.8:5007/NAS)

    疑难问题解决¶

    QNAP无法删除文件¶

    1. 启用admin账户，开启SSH登录，登陆进去
    2. chmod u+w usr/etc/sudoers 开启写权限 (这一步好像每必要)
    3. Vim /usr/etc/sudoers
    4. 找到 admin ALL=(ALL)ALL下面添加mdeng1985 ALL=(ALL)ALL
    5. chmod u-w usr/etc/sudoers 撤销写权限 (这一步好像每必要) 然后ssh mdeng1985账号登录进去 sudo rm -rf某个文件夹即可。
